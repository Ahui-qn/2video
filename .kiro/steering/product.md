# Product Overview

**Script-to-Storyboard AI Tool** (剧本分镜拆解)

A collaborative web application for converting film/TV scripts into detailed storyboards using AI. The tool analyzes scripts to extract characters, assets, and scenes, then generates shot-by-shot breakdowns with visual descriptions optimized for AI image generation.

## Key Features

- **Multi-episode script management**: Upload and organize scripts by episodes
- **AI-powered analysis**: Extracts characters, locations, props, and generates detailed shot breakdowns
- **Asset library**: Maintains consistent character and location descriptions across episodes
- **Real-time collaboration**: Multiple users can work on projects simultaneously via WebSocket
- **Role-based access**: Admin, editor, and viewer roles with different permissions
- **Manual editing**: Full control to add, edit, or delete shots and scenes
- **Export functionality**: Export storyboards to CSV format

## Target Users

Film directors, producers, and pre-production teams who need to break down scripts into actionable storyboards for production planning.
